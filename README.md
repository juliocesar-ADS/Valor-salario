📌 Descrição

Este é um programa simples em Python que solicita o nome e o salário do usuário e classifica sua faixa salarial com base em valores pré-definidos.

⚙️ Funcionalidades
Solicita o nome do usuário
Solicita o salário
Exibe o salário informado
Classifica o usuário em uma das categorias:
💼 CLT (até 3000)
📊 Na média (entre 3001 e 10000)
💰 Rico (acima de 10000)
